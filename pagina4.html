Información útil sobre HTML, CSS y desarrollo web

HTML (HyperText Markup Language) le da estructura a la página web, define los elementos como encabezados, párrafos, imágenes, enlaces, etc. Mientras que CSS (Cascading Style Sheets) le da estilo o apariencia a la página, controlando colores, fuentes, espaciado, diseño y adaptabilidad.

Una propiedad CSS es un atributo que controla un aspecto específico del estilo de un elemento HTML. Por ejemplo, text-align sirve para alinear el texto dependiendo de dónde se especifique.

.texto-centrado {
    text-align: center;
}

.texto-derecha {
    text-align: right;
}

.texto-justificado {
    text-align: justify;
}

La propiedad background-color sirve para cambiar todo el fondo del componente a un color sólido. Se puede especificar usando nombres de colores, códigos hexadecimales, RGB o HSL.

.mi-div {
    background-color: #3498db;  /* Azul usando hexadecimal */
}

.otro-div {
    background-color: rgb(46, 204, 113);  /* Verde usando RGB */
}

.tercer-div {
    background-color: hsl(0, 78%, 60%);  /* Rojo usando HSL */
}

Un layout en CSS es una estructura que define cómo se visualizan los elementos en una página web. Existen varias maneras de crear layouts:

  • Flexbox: Ideal para diseños unidimensionales (filas o columnas).
  • CSS Grid: Perfecto para diseños bidimensionales (filas y columnas).
  • Float: Método más antiguo, usado para posicionar elementos a la izquierda o derecha.
  • Positioned Layout: Usando propiedades como position, top, left, etc.
  • Frameworks: Como Bootstrap, que proporciona un sistema de grid predefinido.

Ejemplo de un layout básico con Flexbox:

.container {
    display: flex;
    justify-content: space-between;
}

.sidebar {
    flex: 1;
}

.main-content {
    flex: 3;
}

Existen varias herramientas que facilitan la generación de código CSS:

Tailwind CSS

Framework de utilidad que permite construir diseños directamente en el HTML mediante clases predefinidas.

Bootstrap

Framework CSS completo que proporciona componentes prediseñados y sistema de grid responsive.

Sass/SCSS

Preprocesador que extiende CSS con variables, mixins y otras funcionalidades.
